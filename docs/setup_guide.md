## 💻 飞书客户端内嵌查看（直接在飞书里看）

为了在飞书客户端里查看排版美化后的看板，请使用以下**仪表盘网页嵌入方案**：

### 第一步：启动数据自动同步服务器
打开终端，运行本地服务器脚本（该脚本会在后台自动拉取飞书最新数据）：
```bash
python3 scripts/server.py
```
*启动后，会输出您的局域网访问地址，形如 `http://192.168.3.15:8080`。*

### 第二步：在飞书多维表格中配置网页组件
1. 打开您的 [飞书多维表格](https://my.feishu.cn/base/E4Aub48pUadmyBskPEjcML63nrh)。
2. 点击表格标签栏右侧的 **“+”** 号 ➔ 选择 **“新建仪表盘”**。
3. 在右侧组件库中，选择 **“网页”** 组件并拖拽入画布中。
4. 在网页 URL 输入框中，填入第一步获取的**局域网访问地址**（例如 `http://192.168.3.15:8080`），保存。
5. **搞定！** 您和同一局域网下的同事现在可以直接在飞书里查看、使用和刷新这个经过极客排版优化后的精美看板了！

## 📂 多维表格架构设计

应用内包含 **4 张核心数据表**，并分别为其创建了直观的**看板视图**与**甘特图视图**：

### 1. 📋 任务看板 (`tblxj1B7MGMgU7Dj`)
用于团队每日任务的可视化跟进、优先级排列及延期警示。
- **主视图**：`任务进度看板` (按“任务状态”分组的看板)
- **甘特图**：`任务甘特图` (按截止日期排期的甘特图)
- **核心字段**：所属模块、任务状态、优先级、负责人、截止日期、延期天数、关联达人。
- **协同方式**：可以直接在看板中拖拽任务卡片调整状态。超期任务建议调整状态为 `已延期`，并补全延期天数。

### 2. 📦 物料管理 (`tbligeEr1TQzKBau`)
物料生命周期流水线跟进（此版已根据最新反馈**去掉拍摄环节**，以达人提供素材后的团队剪辑与发布为主）。
- **流程链路**：
  `待接收素材` ➔ `素材已到` ➔ `剪辑中` ➔ `封面优化中` ➔ `标题优化中` ➔ `待审核` ➔ `已发布`
- **主视图**：`物料状态流水线` (看板视图)
- **甘特图**：`发布甘特图` (甘特图视图，展示素材接收到预计发布排期)
- **核心字段**：剪辑完成、封面完成、标题简介完成、审核通过（均为复选框，方便确认各环节就绪）。

### 3. 👤 达人管理 (`tblhv1jMCaVmeXXe`)
KOL 合作建联及档案库管理。
- **主视图**：`合作状态看板` (按“合作状态”分组的看板，跟踪建联到合作结束的全生命周期)
- **核心字段**：平台、粉丝量、合作类型、合作费用、对接人、沟通记录、交付物料统计。

### 4. 💰 预算管控 (`tblpYwoTY9dH5lX9`)
前置预算执行风险预警，支持**目标-动作-成本**强绑定映射。
- **主视图**：`预算状态预警` (看板视图，直观展示消耗情况，红黄绿预警)
- **核心字段**：预算金额、实际花费、消耗率、关联目标、目标指标、对应动作、预警状态。
- **规则嵌入**：
  - 消耗率在 80%~95% 之间 ➔ `⚠️黄色预警(≥80%)`
  - 消耗率在 95%~100% 之间 ➔ `🔴红色预警(≥95%)`
  - 消耗率超过 100% ➔ `🚨已超支`
