# 📋 飞书「KOL 达人合作管理」模板适配指南

如果您希望直接在飞书多维表格模板中心使用官方模板，并进行微调以完美适配我们的需求，推荐选用飞书官方的 **「KOL 达人合作管理」** 模板（在多维表格模板中心搜索“**达人**”或“**KOL**”即可找到并启用）。

启用该模板后，您可以通过以下 **4 个步骤的微调**，使其完全适配我们的“**达人提供素材 ➔ 团队剪辑 ➔ 审核发布 ➔ 预算管控**”工作流。

---

## 🛠️ 模板微调与适配步骤

### 第一步：适配「物料管理」工作流（去掉拍摄，保留剪辑与审核）
在模板的 **「合作项目跟进表」**（或物料/视频跟进表）中：

1. **修改“物料状态/视频进度”字段（单选）**：
   将原有的状态选项修改为我们设计的 7 步无缝流水线：
   > `待接收素材` ➔ `素材已到` ➔ `剪辑中` ➔ `封面优化中` ➔ `标题优化中` ➔ `待审核` ➔ `已发布`
2. **新增“质检/环节就绪”复选框**：
   新增 4 个复选框字段，用于团队各环节协同确认：
   - `[ ] 剪辑完成`
   - `[ ] 封面完成`
   - `[ ] 标题简介完成`
   - `[ ] 审核通过`
3. **新建“物料状态流水线”看板视图**：
   新建一个看板视图，分组字段选择 **“物料状态/视频进度”**，这样团队就能拖拽卡片推进物料了。

---

### 第二步：适配「任务看板」模块（运营与沟通任务）
在模板中，如果需要一个独立的任务看板来跟进每天的琐碎任务（如：与达人沟通、合同盖章、发票处理等）：

1. **新建「📋 任务看板」数据表**：
   如果模板中没有独立的任务表，建议直接新建一张表，包含以下字段：
   - `任务标题` (文本)
   - `所属模块` (单选：达人沟通 / 达人管理 / 物料管理 / 运营推广)
   - `任务状态` (单选：待处理 / 进行中 / 待审核 / 已完成 / 已延期)
   - `优先级` (单选：P0-紧急 / P1-高 / P2-中 / P3-低)
   - `负责人` (人员)
   - `截止日期` (日期)
   - `延期天数` (数字，用于延期警示)
   - `关联达人` (单向关联「达人资源库」)
2. **新建“进度看板”视图**：
   新建一个看板视图，按“**任务状态**”进行分组，方便拉拽。

---

### 第三步：适配「预算与目标-动作映射」（费用管控）
在模板的 **「预算与收支明细」**（或收支明细表）中，原版模板通常只记录“金额”和“报销人”。我们需要增加**目标-动作-成本强绑定**字段：

1. **新增“映射关系”字段**：
   - **关联目标** (文本，例如：引流加粉转化 / 播放量突破)
   - **目标指标** (文本，例如：粉丝量门槛 1W / 播放量≥10W)
   - **对应动作** (文本，例如：美妆专栏推荐 / 视频合集投放)
   - **预警状态** (单选，选项为：`正常`、`⚠️黄色预警(≥80%)`、`🔴红色预警(≥95%)`、`🚨已超支`)
2. **修改“消耗率”计算**：
   - 实际花费 / 预算金额（可以用公式字段或数字字段）。

---

### 第四步：同步配置自定义大屏看板（可选）
如果您完成了飞书模板的微调，且想继续使用我们的**自定义高颜值暗黑大屏**，只需要：

1. 查看您微调后的数据表在飞书里的 **Table ID**（例如 `tblxxxx`）。
2. 在本地项目的 `scripts/fetch_data.py` 中，将对应的 `KOL_TABLE_ID`、`TASK_TABLE_ID` 等变量修改为新表的 ID。
3. 运行 `./refresh.sh`，大屏就会自动读取并显示您从官方模板修改后的最新数据！
